# Class-2: FastAPI Todo Application

A simple FastAPI application demonstrating basic REST API endpoints for a todo task manager.

## Features
- GET `/todos` - Retrieve all todo items
- GET `/tasks/{task_id}` - Retrieve a specific task by ID

## Requirements
- Python >= 3.13
- FastAPI
- Uvicorn
